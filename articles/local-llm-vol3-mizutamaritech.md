## 「AIに$10稼いでもらおう」という実験

話は少し前に遡る。

Openclawを動かすにはAPIを使う。面白すぎて動かしていると無制限にお金がかかる。今は趣味程度なので自制して月に$10と決めて触っている。（$10なんかは一瞬で吹き飛ぶ）

## AIとの分業

方針が決まった。記事を書く。GitHubで管理してZennで公開する。

でも、どうやって？

Openclawはコンテナの中で動いている。vol.2で話した箱の中だ。その箱の外——私のWSL2環境——にはセキュリティの懸念上、直接触れないようにしている。

最初、Openclawに「GitHubにpushして」と頼んだ。

**できなかった。**

コンテナ内からWSL2のホームディレクトリに直接アクセスできないのだ。当然といえば当然だ。セキュリティのための隔離が、ここでは壁になった。

Openclawが提案してきた。

「PATをDMで渡してもらえればコンテナ内からpushできます。または、ファイルをdocker cpでコピーして自分でpushする方法もあります。」

私は後者を選んだ。**何が起きているか、自分で確認したかったから。**

---

## PATという呪文、そしてハマった

GitHubへのpushにはPAT（Personal Access Token）という認証キーが必要だった。

生成して、貼り付けた。

**エラーが出た。**

原因は単純だった。手順書に書いてあった `[]` をそのまま残してPATを入力していた。`[]`ごと置き換えるのが正しかった。

人間は思い込む。`[]`は「ここに入力」という意味だと頭ではわかっていても、手が止まる。

結局、GitHub CLI（`gh`）をインストールしてブラウザ認証に切り替えた。ターミナルにURLが表示され、ブラウザで承認するだけ。PAT不要になった。これが正解だった。

---

## docker cp という橋

Openclawがコンテナ内に記事ファイルを生成してくれた。

でも私のWSL2からは見えない。

ここで使うのが `docker cp` だ。コンテナとWSL2の間にファイルを橋渡しするコマンドだ。

```bash
docker cp clawdbot-openclaw-gateway-1:/home/node/.openclaw/workspace/zenn-articles/articles/local-llm-vol2.md ~/zenn-articles/articles/
```

コンテナIDを確認して、パスを指定して、コピー先を指定する。

最初は呪文に見えた。でも構造がわかれば読める。**「この箱の中のこのファイルを、ここに持ってきて」**と言っているだけだ。

## GitHubとZennを繋ぐ

ZennはGitHubリポジトリと連携できる。

リポジトリにファイルをpushするだけで、自動的にZennの記事として反映される。ダッシュボードで「公開する」を押せば世界に出る。

仕組みを知ると単純だ。でも知るまでは不思議だった。

ファイルが。GitHubを経由して。Zennになる。

---

## ブラウザ編集とpushの衝突

一度だけpushが拒否された。

GitHubのブラウザ上でREADMEを直接編集していたのに、それをWSL2のローカル環境に反映させないままpushしようとしたからだ。

ローカルとリモートの「現在地」がずれていた。

解決は `git pull` で最新を取得してからpushするだけだ。でも最初は意味がわからなかった。

**教訓：ブラウザでGitHubを直接編集したら、次に作業する前に必ず `git pull` する。**

些細なことだ。でも詰まった。こういうことが積み重なって、少しずつわかっていく。

---

## スラッグという落とし穴

Zennにはスラッグという概念がある。記事ファイルの名前がそのままURLになる仕組みだ。

そしてZenn全体でスラッグが重複すると、デプロイが失敗する。

最初のファイル名が他の誰かと被っていた。

解決策はシンプルだった。ファイル名の末尾に自分のユーザー名を追加する。

`local-llm-vol3-mizutamaritech.md`

自分の名前が入ったファイルが、世に出る。少しだけ、実感が湧いた。

## この記事の流れ、全部

整理すると、こうだ。

```
Openclawが記事を生成（コンテナ内）
　↓
docker cp でWSL2にコピー
　↓
git add / commit / push でGitHubへ
　↓
Zennが自動でデプロイ
　↓
ダッシュボードで「公開」
```

全自動ではない。各ステップで私が確認して、実行している。

それでいいと思っている。今は。
